## **2. 🔒 新增：SSL/TLS 设置与证书管理**

实现完整的SSL/TLS设置页面和证书管理功能。

**功能特性**

| 功能 | 说明 |
|------|------|
| **SSL加密模式** | 支持加密模式选择（灵活/严格/完全/始终使用HTTPS） |
| **最低TLS版本** | 支持选择最低TLS版本 |
| **始终使用HTTPS** | 开关控制是否强制HTTPS访问 |
| **自动HTTPS重写** | 自动重写网页中的HTTP链接为HTTPS |
| **TLS 1.3** | 支持启用/禁用TLS 1.3协议 |
| **通用SSL开关** | 支持切换通用SSL证书状态 |
| **证书包列表** | 显示所有证书包，支持删除自定义SSL证书 |
| **状态展示优化** | 本地化显示证书类型名称和状态 |
| **乐观更新** | 设置修改时立即应用，API失败则回退 |
| **并行加载** | 多项设置并行读取，提升加载速度 |

**调用 API**
- `GET /zones/{zone_id}/settings/ssl` — 获取加密模式
- `GET /zones/{zone_id}/settings/always_use_https` — 获取始终使用HTTPS设置
- `GET /zones/{zone_id}/settings/auto_https_rewrite` — 获取自动HTTPS重写设置
- `GET /zones/{zone_id}/settings/minimum_tls_version` — 获取最低TLS版本
- `GET /zones/{zone_id}/settings/tls_1_3` — 获取TLS 1.3设置
- `GET /zones/{zone_id}/ssl/certificate_packs` — 获取证书包列表
- `GET /zones/{zone_id}/ssl/universal/settings` — 获取通用SSL设置
- `PATCH /zones/{zone_id}/ssl/universal/settings` — 更新通用SSL设置
- `DELETE /zones/{zone_id}/ssl/certificate_packs/{pack_id}` — 删除证书包

## **3. 🔄 新增：转换规则管理**

实现完整的Cloudflare转换规则管理功能，支持URL重写、请求头/响应头修改。

**功能特性**

| 功能 | 说明 |
|------|------|
| **URL重写** | 支持重写路径和查询字符串 |
| **请求头修改** | 支持设置、添加、移除请求头 |
| **响应头修改** | 支持设置、添加、移除响应头 |
| **规则列表** | 按phase分区显示规则（URL重写/请求头修改/响应头修改） |
| **增删改查** | 完整的规则管理操作 |
| **启停开关** | 支持单独启用/禁用规则 |
| **界面本地化** | 所有界面文本适配为简体中文 |

**调用 API**
- `GET /zones/{zone_id}/rulesets/phases/{phase}/entrypoint` — 获取转换规则集（phase: http_request_transform）
- `PUT /zones/{zone_id}/rulesets/phases/{phase}/entrypoint` — 创建规则集入口
- `POST /zones/{zone_id}/rulesets/{ruleset_id}/rules` — 向规则集追加规则
- `PATCH /zones/{zone_id}/rulesets/{ruleset_id}/rules/{rule_id}` — 更新/启停规则
- `DELETE /zones/{zone_id}/rulesets/{ruleset_id}/rules/{rule_id}` — 删除规则

## **4. 📦 新增：缓存规则与速率限制规则**

实现完整的Cloudflare缓存规则与速率限制规则管理页面。

**功能特性**

| 功能 | 说明 |
|------|------|
| **缓存规则管理** | 支持缓存规则的列表查看、新增、编辑、删除 |
| **TTL模式选择** | 支持遵循源站、覆盖为固定值、源站无指令则不缓存 |
| **可缓存开关** | 控制规则是否可缓存 |
| **高级设置检测** | 含高级设置的规则显示只读提示 |
| **速率限制规则** | 支持速率限制规则的完整管理 |
| **周期选项** | 支持10秒/1分钟/10分钟/1小时 |
| **动作选项** | 支持封锁/托管质询/JS质询/记录 |
| **表单校验** | 表达式非空且请求数>0 |

**调用 API**
- `GET /zones/{zone_id}/rulesets/phases/http_request_cache_settings/entrypoint` — 获取缓存规则集
- `GET /zones/{zone_id}/rulesets/phases/http_request_ratelimit/entrypoint` — 获取速率限制规则集

## **5. 📧 新增：邮件路由功能**

实现完整的Cloudflare邮件路由功能界面。

**功能特性**

| 功能 | 说明 |
|------|------|
| **总开关控制** | 支持启用/禁用Email Routing |
| **路由规则管理** | 支持添加、删除路由规则 |
| **目的地址管理** | 支持添加、验证、删除目的地址 |
| **验证状态显示** | 显示地址验证状态（已验证/待验证） |

**调用 API**
- `GET /zones/{zone_id}/email/routing/rules` — 获取路由规则
- `POST /zones/{zone_id}/email/routing/rules` — 添加路由规则
- `PUT /zones/{zone_id}/email/routing/rules/{rule_id}` — 更新路由规则
- `DELETE /zones/{zone_id}/email/routing/rules/{rule_id}` — 删除路由规则
- `GET /accounts/{account_id}/email/routing/addresses` — 获取目的地址（账户级）
- `POST /accounts/{account_id}/email/routing/addresses` — 添加目的地址（账户级）
- `DELETE /accounts/{account_id}/email/routing/addresses/{address_id}` — 删除目的地址（账户级）
