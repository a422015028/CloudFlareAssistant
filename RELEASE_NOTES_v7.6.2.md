**CloudFlare Assistant v7.6.2 更新日志**
========================================

## **1. 🗑️ 新增：Pages 项目旧部署批量清理**

Pages 页面新增「管理部署」功能，支持批量清理账号下所有 Pages 项目的旧部署记录。

**功能特性**

| 功能 | 说明 |
|------|------|
| **批量清理所有项目** | 一键清理账号下所有 Pages 项目的旧部署，每个项目保留指定数量的最新部署 |
| **清理单个项目** | 支持选择指定项目进行清理，保留指定数量的最新部署 |
| **自定义保留数量** | 默认保留 10 个最新部署，可自定义保留数量 |
| **清理结果详情** | 清理完成后显示每个项目的清理情况及总计清理数量 |
| **二次确认** | 清理前弹出确认对话框，防止误操作 |

**调用 API**
- `GET /accounts/{account_id}/pages/projects/{project_name}/deployments` — 列出部署
- `DELETE /accounts/{account_id}/pages/projects/{project_name}/deployments/{deployment_id}` — 删除部署

***

**功能特性**

| **清理单个脚本** | 支持选择指定脚本进行清理，保留指定数量的最新版本 |
| **自定义保留数量** | 默认保留 10 个最新版本，可自定义保留数量 |
| **清理结果详情** | 清理完成后显示每个脚本的清理情况及总计清理数量 |

***

## **3. 🎨 优化：部署卡片折叠与 UI 统一**

统一 Pages 和 Worker 部署卡片的视觉风格与交互体验。

**功能特性**

| 功能 | 说明 |
|------|------|
| **卡片可折叠** | Pages 和 Worker 部署卡片改为可折叠样式，点击标题栏展开/收起 |
| **状态持久化** | 卡片展开/折叠状态通过 SharedPreferences 保存，重启应用后恢复 |
| **默认折叠** | 首次打开默认折叠状态，减少页面占用空间 |
| **尺寸统一** | Pages 和 Worker 卡片内边距、按钮高度、文字大小统一 |

***

## **4. 🎨 优化：列表标题栏统一布局**

统一 Pages 项目列表和 Worker 脚本列表的标题栏布局结构。

**功能特性**

| 功能 | 说明 |
|------|------|
| **统一按钮布局** | 标题行统一为：标题 + 管理项目/脚本 + 管理部署/版本 + 刷新 |
| **操作栏折叠** | 点击「管理项目/脚本」展开操作栏（全选、批量删除），再次点击收起 |
| **刷新按钮靠右** | 刷新按钮始终靠右对齐，不受左侧按钮变化影响 |
